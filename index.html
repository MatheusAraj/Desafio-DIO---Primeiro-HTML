Definição de Client e Servidor

A comunicação funciona da seguinte forma, o cliente faz um pedido. o servidor recebe esse pedido, processa e responde, o cliente recebe e exibe o conteudo, a comunicação utiliza o protocolo http, ele define como os dados sao troados entre cliente e servidor
Client é quem faz o pedido.
Exemplo: Seu navegador (Chrome, Firefox etc.) quando você acessa um site.

  • É o dispositivo do usuário(celular, computador, etc.)
  • Envia solicitações ao servidor.
  • Recebe e exibe os dados (ex: o site que você pediu.)
Servidor é quem responde ao pedido.
Exemplo: Um computador na internet que "entrega" a página do Google quando você acessa google.com.
  • Armazena sites, sistemas, bancos de dados, arquivos etc.
  • Processa os pedidos dos clientes.
  • Envia os dados de volta ao cliente.


O que é uma IDE?

Uma IDE (Ambiente de Desenvolvimento Integrado) é um software que reúne diversas ferramentas essenciais para facilitar e organizar o desenvolvimento de código. No curso "Primeiros Passos com HTML", utilizamos o VS Code, uma IDE licenciada pela Microsoft, reconhecida por sua leveza, versatilidade e ampla gama de extensões que tornam o processo de programação mais eficiente e produtivo.


Estruturando seu HTML

Estruturar bem o HTML é essencial para manter o código organizado, facilitar a manutenção e garantir que o conteúdo seja acessível por todos, inclusive por leitores de tela. Uma boa estrutura também melhora o SEO, ajuda no uso de CSS e JavaScript e segue os padrões da web, garantindo que o site funcione corretamente em diferentes dispositivos e navegadores. Além disso, um código bem feito é mais fácil de entender e evoluir.

Tags de estrutura básica - <HTML> <HEAD> <BODY>
Textos - <H1> <H2> <H3> <P>
Lista ordenadas e Não ordenadas - <OL> <UL> <LI>
Links - <A> heref="youtube.com"> <A>
Um site muito utilizado para estudo de HTML é o w3schools, onde esta presente a lista de todas as tags do html


Trabalhando com Formularios

Trabalhar com formulários em HTML é fundamental para coletar dados dos usuários, como cadastros, pesquisas e comentários. Eles permitem a interação entre o usuário e o site, sendo essenciais para sistemas dinâmicos. Além disso, formulários bem feitos melhoram a experiência do usuário, facilitam a validação de dados e a integração com servidores e bancos de dados.

Tags - <FORM> </FORM> Onsubmit="" Method="get">

Checkbox e Radio - <Input type="checkbox" name="exemplo" value="exemplo">
Button e seus tipos - <button type="submit">Enviar</button>
Select e seus tipos - <Select name="role"> <option value "">Selecione o cargo</option> </select>
Textarea - <textarea name="message"></textarea>


Formatação

A formatação em HTML é essencial para organizar e destacar o conteúdo da página, tornando a leitura mais clara e agradável. Usar corretamente tags como >strong>, >em>, >p>, e listas ajuda a transmitir a informação de forma estruturada e compreensível. Além disso, uma boa formatação contribui para a acessibilidade e melhora a apresentação do site em diferentes dispositivos.

Formatação de textos - <strong <b> <i> <u> <sup> <br>
Mais formatações - <font color="red" face="arial, sans, thorome, etc...">Texto exemplo</font>
DIV E SPAN - <div id=""> <div name=""> <span>
Fieldset - <fieldset> <legend> </fieldset>
Embeds e Iframes - <iframe src="....">
Resenha sobre cores - RGB HEX HSL RGBA HSLA

Além disso, o uso adequado de mídias pode reforçar o entendimento, engajar visitantes e valorizar a aparência do site, tornando-o mais moderno e acessível.

Criando Tabelas

As tabelas em HTML são importantes para organizar e exibir dados de forma clara e estruturada, facilitando a leitura e a comparação de informações. Elas são ideais para apresentar listas, horários, preços, resultados e qualquer conteúdo que exija organização em linhas e colunas. Quando bem utilizadas, melhoram a usabilidade e tornam o site mais profissional e informativo.
Tag table - <table>Define o espaço da Tabela</table> Border, Width, cellspacing, align, cellpadding, summary
Tag TR - <tr> Table row (Linha da tabela) ID
Tag TH - <th tittle="Nome completo" Nome </th> Define o cabeçalho da tabela
Tag TD - Célula comum
Tag THEAD - Definir o cabeçalho <thead> </thead>
Tag TBODY - Estruturar a tabela, usado como principal <tbody> </tbody>
Tag TFOOT - define o rodapé <tfoot> </tfoot>


HTML Semantico

O HTML semântico é importante porque utiliza tags com significado claro, como <header>, <nav>, <main>, <section> e <footer>, que indicam a função de cada parte do conteúdo. Isso melhora a organização do código, facilita o trabalho de desenvolvedores, aumenta a acessibilidade para tecnologias assistivas e ajuda os mecanismos de busca a entenderem melhor o conteúdo, beneficiando o SEO.
